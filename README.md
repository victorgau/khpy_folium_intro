* [folium 說明文件](http://python-visualization.github.io/folium/)
* [GeoPandas](https://geopandas.org/)

副檔名含意：
* .dbf ==> 存放數值 (number) 及文字 (text) 資料
* .shp ==> 存放形狀 (shapes) 及位置 (positions) 資料
* .shx ==> 存放索引(index)
* .prj ==> 存放投影(projection)資訊
